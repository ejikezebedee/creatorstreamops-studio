# Platform Boundaries

CreatorStreamOps Studio supports planning, documentation, reporting, and manual tracking only.

Forbidden functionality:

- Auto-like bots
- Auto-comment bots
- Follower generators
- View bots
- Fake gift manipulation
- Mass DM automation
- Unofficial TikTok scraping
- Hidden browser automation
- TikTok LIVE eligibility bypass
- Copyright bypassing or copyrighted media downloading
- Credential theft or TikTok password collection

Future integrations must use official APIs where permitted, must be documented, and must require explicit user approval.
