# Product Scope

CreatorStreamOps Studio is a productivity and reporting tool for creator operations.

## Included

- Local content planning
- Script and caption drafting
- Hashtag organization
- Content calendar
- TikTok LIVE preparation
- Human moderation planning
- OBS / TikTok LIVE Studio checklist
- Manual analytics tracking
- TikTok Shop content and LIVE-selling planning
- Markdown reports
- Audit logging

## Excluded

- Real TikTok posting in the MVP
- Credential storage for TikTok
- Unofficial data scraping
- Fake engagement automation
- Downloading copyrighted content
- Bypassing platform eligibility or rules
