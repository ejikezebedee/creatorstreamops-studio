# Roadmap

## MVP

- Local admin authentication
- Local JSON storage with atomic writes
- Video, script, caption, calendar, LIVE, moderation, analytics, and report modules
- Manual-only analytics
- Platform-boundary documentation and secret scan

## Next

- SQLite storage option
- CSV export and import
- Richer calendar views
- PDF export
- Role-based local users
- Official TikTok API integration research, only where documented and allowed

## Not Planned

- Fake engagement automation
- Unofficial scraping
- Mass messaging
- Credential collection
- Copyright bypass tools
